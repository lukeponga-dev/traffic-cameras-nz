# SpeedWatch Mobile

SpeedWatch Mobile is a React Native application for iOS and Android that functions as a speedometer and location tracker. It's designed to provide users with real-time speed information and to log their location history.

## Features

*   **Real-time Speedometer:** Displays the user's current speed using the device's GPS.
*   **Interactive Map:** Shows the user's current location on a map, which updates in real-time.
*   **Location History:** Automatically records the user's location data, including coordinates and timestamps.
*   **History View:** Users can view their logged location history in a clear, scrollable list.
*   **Data Management:** Functionality to clear the entire location history is provided.
*   **Cross-Platform:** Built with React Native for a consistent experience on both iOS and Android.
