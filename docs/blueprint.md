# **App Name**: SpeedWatch

## Core Features:

- Interactive Map: Display speed camera locations on an interactive map with color-coded pins based on camera type and status.
- Camera Details: Show detailed information about each speed camera, including type, status, and location, when a user taps on a pin.
- Filtering: Allow users to filter cameras by type (fixed, average, inactive) and status.
- User Location Tracking: Track user's current location on the map.
- Reporting System: Enable users to report new cameras, status changes, data errors, and location updates, with location capture for accurate reporting. After verifying by human eyes, report details are used as a tool to update data and increase overall app accuracy.
- Settings: App settings that affect user experience (such as theme selection) can be altered in this module.
- Cloud SQL DB Sync: Synchronize data from the Cloud SQL database into the application for use on the interactive map.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), evocative of vigilance, alertness, and authority. It's a professional and calming hue that references the purpose of speed cameras.
- Background color: Light gray (#F0F2F5), providing a neutral backdrop that ensures readability and focus on the map and data.
- Accent color: Bright orange (#FF9800), drawing attention to CTAs and important notifications.
- Body and headline font: 'Poppins' (sans-serif) for a contemporary and readable user interface.
- Use clear and intuitive icons to represent different camera types and report options.
- A clean, map-centric layout is implemented for immediate access to interactive location details.
- Smooth transitions and animations on map interactions to create a fluid and responsive user experience.
